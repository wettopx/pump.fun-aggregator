Classic Launches

The traditional way to create tokens, but with the disk.fun twist. Classic launches give you full control over your token's identity.

How It Works

  1. Connect Wallet: Link your Phantom wallet to get started
  2. Choose Classic: Select the Classic tab in the launcher
  3. Enter Details:
    • Token Name (up to 32 characters)
    • Token Symbol (up to 10 characters)
    • Upload Image (JPG/PNG, max 5MB)
    • Add social links (optional)
    • Set dev buy amount (optional, max 2 SOL)
  4. Pay Fee: 0.04 SOL service fee + gas
  5. Launch: Your token goes live on pump.fun instantly

AI Name Generation

Infected token names are generated using Hugging Face AI with intelligent uniqueness checking:

The dev buy feature allows token creators to purchase an initial amount of their own token at launch, demonstrating commitment and providing initial liquidity.

How Dev Buy Works

  1. Available only for Classic tokens
  2. Set amount during token creation (0-2 SOL)

Benefits of Dev Buy
